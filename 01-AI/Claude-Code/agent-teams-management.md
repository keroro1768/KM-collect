# AI 團隊管理 - Claude Code Agent Teams

- 管理文章: https://www.managertoday.com.tw/columns/view/71884

## 核心概念

Claude Code 推出的實驗性功能 **Agent Teams**，核心邏輯：

> 讓開發者從「給單個 AI 寫提示詞的執行者」，轉變為「管理 AI 研發團隊的技術負責人」

## 角色分工

| 角色 | 說明 |
|------|------|
| **Team Lead (團隊負責人)** | 由初始 Claude Code 會話擔任，統籌整個團隊 |
| **Agent 團隊成員** | 多個獨立的 Claude Code 實例，各司其職 |

## 團隊負責人職責

- 創建團隊
- 指派初始任務
- 設置任務依賴
- 合成最終結果
- 手動參與代碼編寫

## 優勢

1. **並行協作** - 多個 AI Agent 同時工作
2. **角色分工** - 不同 Agent 擔任不同角色
3. **自主調度** - 跨角色的信息同步與任務調度

## 應用場景

- 複雜專案開發
- 大型代碼重構
- 多模組系統整合
- 研發流程自動化
