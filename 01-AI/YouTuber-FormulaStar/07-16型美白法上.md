## 📌 影片摘要

### 為什麼會變黑？

黑色素形成過程：
1. **刺激** → 紫外線、發炎、賀爾蒙
2. **活化** → 酪胺酸酶被激活
3. **合成** → 黑色素細胞產生黑色素
4. **轉移** → 黑色素從基底層往上推
5. **沉澱** → 形成色斑、變黑

---

## 🟤 16種變黑體質類型

根據原因分類：

### 1. 曬黑型
- 原因：紫外線
- 解決：防曬 + 維C

### 2. 發炎型
- 原因：痘痘、傷口、摩擦
- 解決：先抗炎

### 3. 賀爾蒙型
- 原因：懷孕、服用避孕藥
- 解決：傳明酸

### 4. 壓力型
- 原因：壓力、失眠
- 解決：抗氧化

### 5. 糖化型
- 原因：攝取過多糖
- 解決：抗糖化

### 6. 氧化型
- 原因：自由基
- 解決：抗氧化

### 7. 基因型
- 原因：天生的
- 解決：只能淡化，無法完全消除

## ⚠️ 常見美白錯誤

| 錯誤 | 後果 |
|------|------|
| 只用美白產品 | 不防曬 = 無效 |
| 過度去角質 | 變敏感 |
| 乱用成分 | 反而變黑 |
| 期待太快 | 美白需要時間 |

---

## ✅ 正確美白步驟

1. **防曬** - 一切的根本
2. **找原因** - 先確定為什麼變黑
3. **對症下藥** - 不同原因用不同成分
4. **坚持** - 至少28天（皮膚代謝週期）
