# M487 資料下載指南

## Datasheet 下載（需登入）

Nuvoton 官網下載區需要 Nuvoton 會員帳號：

### 直接下載連結（需登入）
- **M487 Datasheet**: https://www.nuvoton.com/resource-download.jsp?tp_GUID=DS1320231124075819
- **M487 User Manual**: https://www.nuvoton.com/resource-download.jsp?tp_GUID=UM1320231124075822

### 下載步驟
1. 前往 https://www.nuvoton.com/
2. 點右上角 `Login` 或 `Member Centre`
3. 註冊或登入帳號
4. 回到下載頁面即可下載

---

## BSP (Board Support Package)

### M487 BSP 下載
- **路徑**: https://www.nuvoton.com/tool-and-software/board-support-package/
- **晶片**: NuMicro M4 → M48x 系列

BSP 包含：
- CMSIS 驅動程式
- 周邊驅動範例
- 評估板範例
- Linker Script
- startup code
